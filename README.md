# الورشة الذكية | Smart Repair Shop

منصة عربية متكاملة لورش صيانة السيارات تجمع بين التشخيص المدعوم بالذكاء الاصطناعي، ترجمة أكواد OBD-II، تتبع الصيانة، سوق قطع الغيار، إدارة الفروع، العقود، والدفع الإلكتروني.

## المزايا

- تشخيص ذكي للأصوات والأعراض مع تقارير بثلاث لغات تشغيلية: النظام، الفني، ومالك السيارة.
- ترجمة أكواد OBD-II وتقدير مستوى الخطورة والتكلفة وقطع الغيار.
- وضع محاكاة تلقائي عند عدم إضافة مفتاح Gemini، لذلك تعمل المنصة مباشرة للتجربة.
- واجهة عربية RTL متجاوبة مبنية بـ React وTailwind CSS.
- خادم Express موحد لتشغيل الواجهة وواجهات API.
- نقطة فحص صحة جاهزة للاستضافة: `/api/health`.

## التشغيل محلياً

```bash
npm install
cp .env.example .env
npm run dev
```

ثم افتح: `http://localhost:3000`

للتشغيل بالذكاء الاصطناعي الفعلي، أضف مفتاح Gemini في ملف `.env`:

```env
GEMINI_API_KEY=your_key_here
```

## الفحص والبناء

```bash
npm run lint
npm run build
npm start
```

## النشر على Render

المشروع يتضمن ملف `render.yaml`. اربط المستودع في Render وأضف متغير البيئة `GEMINI_API_KEY`. سيستخدم Render أوامر البناء والتشغيل ونقطة فحص الصحة تلقائياً.

## المتطلبات التقنية

- Node.js 20 أو أحدث
- Gemini API اختياري؛ تعمل المنصة بدونه في وضع المحاكاة

## تنبيه مهني

نتائج التشخيص الذكي تقديرية ومساندة للفني، ولا تغني عن الفحص الميكانيكي المباشر، خصوصاً في الأعطال الحرجة المتعلقة بالمكابح والتوجيه والمحرك.
